# Shape Gen

Shape Gen is a browser-based Minecraft shape planning tool for generating pixel blueprints, including circles, ellipses, lines, arcs, and rectangles.

## Live Page

Use the deployed app here:

https://chrisofnormandy.github.io/shape-gen/

## Installation

1. Clone the repository:

```bash
git clone https://github.com/chrisofnormandy/shape-gen.git
```

2. Enter the project directory:

```bash
cd shape-gen
```

3. Install dependencies:

```bash
yarn install
```

## Run Locally

Start the development server:

```bash
yarn dev
```

Then open the local URL shown in your terminal (typically `http://localhost:3000`).

## Build

Create a production build:

```bash
yarn build
```
