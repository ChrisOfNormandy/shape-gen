# How to Use

Shape Gen is a Minecraft circle generator and pixel blueprint tool for creating circles, ellipses, lines, rectangles, and arcs.

On the left is a form for creating new shapes.
Select a shape using the option select, then set its definition.

Each shape will have the following options:

- Buffer - an adjustment to how the shape is drawn in curves; a higher value increases the curve size, while a lower value decreases it
- Color - the shape color in the grid
- Layer - the drawing layer of the shape
- Origin X - where the starting point or center of the shape is horizontally
- Origin Y - where the starting point or center of the shape is vertically

For an ellipse, the origin is its center; rectangles are the top-left corner; lines and arcs use it as their starting point.

For an ellipse:

- Radius X - from center to edge horizontally
- Radius Y - from center to edge vertically

For a rectangle:

- Width - the shape's size horizontally
- Height - the shape's size vertically

For a line:

- Endpoint X - where the line ends horizontally
- Endpoint Y - where the line ends vertically

For an arc (bezier curve):

- Endpoint X - where the line ends horizontally
- Endpoint Y - where the line ends vertically
- Control Point X - where the arc's curve is centered horizontally
- Control Point Y - where the arc's curve is centered vertically

To draw the shape, click **Create Shape**.
This will create and draw the shape in the grid workspace.

A list of shapes is shown in a list on the right side.
There is where you may modify each shape with the same inputs used to create them, as well as copy or delete them.

> You can show or hide this list by clicking the **>** button in the bottom right.
> The "shape creation" section can be dragged around the screen, too.
